# @vault/ledger 💰

**Zero-Trust Encrypted Financial Ledger**

A focused, independent module of the Vault Tracker ecosystem designed for private financial tracking and intelligence.

## Features
- **AES-256-GCM Encryption**: All financial records are encrypted locally.
- **Financial Intelligence**: Analytics dashboard with 50/30/20 balance analysis and spike detection.
- **Spending Classification**: Easily categorize entries as Needs vs. Wants.
- **Calendar Integration**: Audit your spending habits across time.

## Getting Started
1. `npm install`
2. `npm run dev`

## Privacy Policy
Your data never leaves your device unencrypted. No tracking, no cloud syncing, 100% private.
